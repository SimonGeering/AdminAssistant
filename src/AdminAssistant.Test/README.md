# AdminAssistant.Test - ReadMe

ALL Commands assume a woking folder of the AdminAssistant.Test csproj file location.

## CLI Commands

Restore tools from `.config/dotnet-tools.json` settings.

> `dotnet tool restore`

Run mutation tests

> `dotnet stryker`
